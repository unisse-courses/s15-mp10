**How to install:**

Run in Terminal
1. run `npm install`
2. create a `.env` in the root with the contents:
    `DBUSER=CCAPDEV`
    `DBPASS=CCAPDEV`
3. run `node app.js`

**DEPLOYMENT:**
1.  Deployed to heroku 
    Can be accessed using this link : https://ccapdevs15mp10.herokuapp.com
